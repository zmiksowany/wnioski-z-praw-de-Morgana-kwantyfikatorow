Wnioski z prawa de Morgana kwantyfikatorów:

  • prawa de Morgana:
    • ∼ (∃x : p(x)) ⇔ ∀x : ∼ p(x)
    • ∼ (∀x : p(x)) ⇔ ∃x : ∼ p(x)

Przykłady:

Zaprzeczeniem zdania:
∀ε > 0 ∃N ∈ N ∀n > N : |an − g| < ε
jest
∃ε > 0 ∀N ∈ N ∃n > N : |an − g| ≥ ε

Natomiast zaprzeczeniem zdania:
∀ε > 0 ∃δ > 0 : (|x − x0| < δ ⇒ |f(x) − g| < ε)
jest
∃ε > 0 ∀δ > 0 : (|x − x0| < δ ∧ |f(x) − g| ≥ ε)

Wnioski:

Zastosowanie prawa de Morgana w tych przypadkach pokazuje, że negację zdań możemy uzyskać poprzez zamianę kwantyfikatorów ∀ ↔ ∃ oraz negację zdań występujących pod tymi kwantyfikatorami.
